## üí° Key Findings Summary

### The Problem You're Solving
Complex websites (Reddit, heavy UI) break extraction because:
- Semantic selectors miss non-standard layouts
- Readability sees multiple content areas (post + comments)
- Need intelligent fallback to content scoring

### The Solution (3-Stage Pipeline)
1. **Stage 1:** Try semantic extraction (article, main tags)
2. **Stage 2:** Try Mozilla Readability
3. **Stage 3:** Score all elements, pick best content (ScoringEngine)
